# Demo Version to use kinect depth info to show a popup alert

## Requirements
Download `yolov3.h5` pretrained model and place it in the `model_data` directory

## Dependencies

`Shreesh-module`
- kivy
- py2neo
- pyKinectAzure
- tensorflow (2.3.x ???)
- keras
- yolo3 modules
- cv2
- PIL
- numpy


`Ozaki-module`
- googletrans==4.0.0-rc1
- pandas
